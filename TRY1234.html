นาย ธัญพิสิษฐ์ หินเดช ชื่อเล่น นนท์ เกิด 12/06/2544 ชื่นชอบการออกแบบ โปรแกรม การทำงาน ชอบหาประสบการใหม่ พร้อมสำหรับการทำงานที่ท้าทายเป็นอย่างมาก

ประสบการการออกงาน เคยแข่งเขียนโปรแกรม Arduino และทำโปรเจต เกี่ยวกับ สัญญาณ Pulse Width Modulation(PWM) ปรับระดับแรงดัน (ปวช)

เคยเป็นพิธีกรงานแข่งหุ่นยน และ เข้าอบลมการเขียนโปรแกรม LabView และทำโปรเจต แบบจำลอง เครื่องตัดไม้โดยใช้โปรแกรม

LabView , PLC , C++ และเขียนโปรแกรม เช็คความถูกต้องของสินค้าโดยใช้โปรแกรม visio (ปวส)

เคยเป็นพิธีกรงาน งาน skill ex pro 2022 ของ บูธอุตสาหกรรม อัตโนมัติ โดยใช้อุปกรณ์ ของ Festo และ เข้างานออกค่ายอบลม เกี่ยวกับอุปกรณ์ เซ็นเซอร์ อุตสาหกรรม และ แบบ จำลอง โรงงานจริง

ทำโปรเจต database โดยใช้ Microsoft SQL Server โดยโปรเจตนี้เป็นการทำต่อของรุ่นพี่ ที่เป็นโปรเจต แขนกล dobot ยก,ย้ายสิ่งของจากสายพานยกไปเก็บในคลัง

กลุ่มผมรับทำต่อคือการทำ database โดยจะนับจำนวน และ ประมวลคุณภาพ ชั้นที่ 4 พีระมิดอัตโนมัติในอุตสาหกรรม ระดับ 4 : Management (การจัดการ) (ป.ตรี)
